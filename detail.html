Machine Learning:
Dasar hingga Implementasi

Deskripsi:

Course ini dirancang untuk memberikan pemahaman mendalam tentang konsep dasar Machine Learning serta penerapannya dalam dunia nyata. Peserta akan mempelajari algoritma-algoritma populer seperti Linear Regression, Decision Tree, K-Nearest Neighbors, dan Neural Networks. Dengan pendekatan praktis, course ini juga mencakup latihan langsung menggunakan Python dan pustaka seperti Scikit-learn dan TensorFlow.

Topik yang Dipelajari:

  • Pengenalan Machine Learning dan jenis-jenisnya
  • Proses preprocessing data
  • Algoritma Supervised & Unsupervised Learning
  • Evaluasi model dan overfitting
  • Implementasi model dengan Python (Scikit-learn & TensorFlow)
  • Studi kasus proyek mini

Level: Pemula hingga Menengah

Prasyarat:

  • Pemahaman dasar Python
  • Logika pemrograman dan matematika dasar (aljabar & statistik)
